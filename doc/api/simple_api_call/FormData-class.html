FormData class

A class for form data request body

e.g.

var formData = FormData(
  fields: {
    'name': 'Raj',
    'age': '30',
    'skills': ['Dart', 'Flutter'],
  },
  files: {
    'songs': ['music.mp3', 'tune.mp3'],
    'image': '../assets/nature.png',
  },
);
